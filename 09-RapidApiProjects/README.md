# Rapid API Projects

These project tought me how to consume API's from Rapid API. I consumed Gas Prices API, Imdb.com API, Booking.com API

## Technologies Used
The following technologies have been used in this project:

- **.NET 8**
- **Rapid API**
- **Bootstrap**
- **MVC**
